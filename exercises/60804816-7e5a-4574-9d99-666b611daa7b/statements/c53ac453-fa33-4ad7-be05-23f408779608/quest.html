Define the Rectangle class such that:

  • the length of both sides ( a and b ) will be set as default to 1.0,
  • its constructor will take two parameters ( a and b ) both of them will define the lengths of the sides of a rectangle,
  • included perimeter and area methods calculating the perimeter and area of a rectangle with sides a and b ,
  • displaying an object of this class would generate the text 'Rectangle ( a , b )' where a and b should be replaced with the side lengths of the rectangle.
Create an object of this class for sides with lengths 3 and 4 and store it in the variable p . Display the contents of this variable, followed by the perimeter and area of the rectangle it contains.
