Problem 8 hetmanów

Create a program that displays a coded representation of a chessboard filled with eight Q's, which represents the specific positions where the Queen (chess) should be placed - see below.

  • Hetman szachuje wszystkie pola w tych samych liniach: poziomej, pionowej i po przekątnych.
  • Standardowe wymiary szachownicy to 8 x 8

Napisz kod, który wskaże w jaki sposób rozstawić na szachownicy 8 hetmanów, tak by żaden nie szachował innego.



Wynik dla 8 hetmanów powinien wyglądać następująco: 


|Q|+|+|+|+|+|+|+|
|+|+|+|+|+|+|Q|+|
|+|+|+|+|Q|+|+|+|
|+|+|+|+|+|+|+|Q|
|+|Q|+|+|+|+|+|+|
|+|+|+|Q|+|+|+|+|
|+|+|+|+|+|Q|+|+|
|+|+|Q|+|+|+|+|+|


Hint: To store the momentary state of the solution, you can create a class Queens. The data structure is crucial. Instead of a two-dimensional 8x8 array, you can use four one-dimensional lists.
