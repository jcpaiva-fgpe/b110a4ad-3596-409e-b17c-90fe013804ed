Write a program that reads some text from the user and counts the number of times each word appears in it. Use a dictionary to store the counter of individual words. Display this dictionary at the end of the program. Don't use the if statement or the count method, instead use the split (string) and setdefault (dictionary) methods, and The + operator.
Start the program and enter the string 'ram ram pam pam ram pam pam ram pam pam'. The expected result wil be: '{' ram ': 4,' pam ': 6}'.
