Zwróć uwagę, jak w poniższym kodzie wykorzystano funkcję range do zainicjalizowania zbioru wartościami z zakresu od 0 do 99.
Uzupełnij jego ostatnią linię, tak by wyświetlała średnią arytmetyczną wartości wszystkich elementów zbioru.

Pomocne zwroty:

'Number of set items' - Ilość elementów w zbiorze.

'Smallest set element' - Najmniejszy element w zbiorze.

'Largest set element' - Największy element w zbiorze.

'Average value of the set' - Średnia wartośc zbioru.
